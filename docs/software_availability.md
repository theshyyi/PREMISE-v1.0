# Software availability note for manuscript use

**Name of software:** PREMISE v1.0

**Description:** PREMISE is an open-source Python framework for basin-oriented preprocessing, harmonization, index generation, and comparative evaluation of multi-source gridded hydroclimatic datasets in environmental modelling workflows. The current release supports standardized ingestion of heterogeneous products, NetCDF-oriented conversion and metadata harmonization, unit and coordinate normalization, basin-scale clipping and aggregation, precipitation-related extreme and drought index calculation, and reproducible product evaluation against reference datasets.

**Developers:** Xinlong Le

**Programming language:** Python

**System requirements:** Linux, Windows, or macOS; recommended environment based on Python 3.10 or above.

**Dependencies:** Core dependencies include xarray, numpy, pandas, netCDF4, geopandas, rasterio, matplotlib, and related geospatial or scientific Python libraries. Some input readers and plotting workflows require optional format-specific dependencies.
