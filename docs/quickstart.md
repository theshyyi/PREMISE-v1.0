# Quick start

The quickest way to verify the repository is to run the synthetic quick-start example:

```bash
python examples/quickstart/basic_usage.py
```

This script creates a small monthly precipitation cube, computes SPI-3, and evaluates a simple RMSE diagnostic. It does not require external data.

A second overview example prints the available source catalogue and module-level workflow description:

```bash
python examples/quickstart/module_overview.py
```
