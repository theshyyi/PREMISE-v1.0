# PREMISE v1.0 module architecture

PREMISE is organized around six application-oriented modules.

1. **Data acquisition** for multi-source hydroclimatic products.
2. **Data conversion and harmonization** for heterogeneous raw formats and metadata.
3. **Basin-oriented preprocessing** for clipping, masking, aggregation, and model-facing forcing preparation.
4. **Product evaluation** for comparative benchmarking against reference datasets.
5. **Extreme climate and drought indices** for hydroclimatic diagnostics.
6. **Visualization and reporting** for figures, summaries, and manuscript-facing outputs.

Legacy modules are retained for backward compatibility, but new examples should prefer the six-module architecture.
