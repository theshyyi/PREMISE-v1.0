# China case-study workflow

This folder is intended for larger manuscript-oriented case studies. In a publication-ready version of the repository, this directory should contain:

- a configuration file describing products, reference data, study period, and target grid,
- a shapefile or instructions for obtaining the analysis regions,
- a workflow script for harmonization,
- a workflow script for comparative evaluation, and
- a short note explaining the provenance of each derived figure or table.

Large raw data files are not bundled with the repository, but the workflow should remain runnable once the user updates the configuration paths.
