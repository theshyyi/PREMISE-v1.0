# Extreme and drought index workflow

This example slot is reserved for unified generation of extreme climate and drought indices.

Recommended steps:

1. Load harmonized precipitation or hydroclimatic variables.
2. Compute drought indices with `premise.climate_indices.drought`.
3. Compute precipitation-extreme indices with `premise.climate_indices.extremes`.
4. Export standardized NetCDF or tabular summaries for downstream diagnostics.
